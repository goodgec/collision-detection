Detailed Description

Custom queue for managing Positions of Particles.

MultiQueue manages Nodes which carry Position information for a single timestep. The underlying data structure is a doubly linked list. In addition to functioning as a normal queue, Nodes in the middle of the MultiQueue may be accessed and dequeued from the queue. This feature is used to pop Nodes of an arbitrary timestep for a given Particle which is required by the CollisionDisplay. MultiQueue also includes its own lock for synchronization.
