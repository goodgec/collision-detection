bool Collision::operator< ( Collision *  other)

Returns true if this Collision's overlap time is longer than the overlap time of the Collision we are comparing to.

This is counter-intuitive, but it allows us to use the built in sorting and end up with a sorted list of collisions where those with the longest overlap times come first (because those get bounced first).

Parameters
otherThe Collision we are comparing this Collision to.
Returns
Whether or not this collision has a longer overlap time.
